The #100DaysofCoding Challenge

Day One Project

Today is the first day I'm attempting the #100DaysofCoding challenge. Or 100DaysofCode, depending on the site and who is running it. For my first project, I'm building a simple website proclaiming that I'm attempting the challenge. This page is using HTML and CSS, but no Javascript. For the first project, I wanted to focus solely on those two languages.

I'm hoping that I'll be able to work some Javascript into other projects. I'd love to learn how to do image art solely using CSS, while learning more about Bootstrap and how it can really make a website 'pop'.

  1. Make it through the 100 days
  2. Build a portfolio of projects on Github
  3. Work with my Twitch community to grow and learn as a coder
  4. Learn more about Javascript and UI/UX development
  5. Learn how to make images using only CSS
  6. Make new friends and connections through Twitch and Twitter
  7. Have fun doing it!

Thank you everyone for your support and your encouragement. This is only the beginning. Even the longest journey begins with a single step. This is that step. Let's see where this journey takes us, together.
